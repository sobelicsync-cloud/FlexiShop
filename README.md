# 🚀 Flexishop - Premium Streetwear & Vapes Store

Bienvenido a **Flexishop**, una plataforma de catálogo digital diseñada para ofrecer una experiencia de compra fluida, moderna y optimizada para dispositivos móviles.

## ✨ Características
- **Diseño Ultra-Moderno:** Interfaz oscura con efectos de "Glassmorphism" (vidrio esmerilado).
- **Catálogo Dinámico:** Filtrado inteligente por categorías (Vapes, Gorras, Sneakers, etc.).
- **Carrito de Compras:** Gestión en tiempo real con persistencia en `localStorage`.
- **Checkout vía WhatsApp:** Generación automática de mensajes con el resumen del pedido para cerrar ventas al instante.
- **Totalmente Responsivo:** Optimizado para navegar desde cualquier smartphone o PC.

## 🛠️ Tecnologías Usadas
- **React.js** (Vite)
- **Tailwind CSS** (Estilos rápidos y modernos)
- **Lucide React** (Iconografía minimalista)
- **Context API** (Gestión de estado del carrito)
